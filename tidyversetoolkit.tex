Installing Tidyverse


Install the complete tidyverse with a single line of code:

install.packages("tidyverse")

Load the core tidyverse packages: ggplot2, tibble, tidyr, readr, purrr, and dplyr. 
These are the packages you are likely to use in almost every analyis.

\begin{frame}[fragile}
\large

\textbf{readr}\\
The goal of readr is to provide a fast and friendly way to read rectangular data (like csv, tsv, and fwf). 
It is designed to flexibly parse many types of data found in the wild, while still cleanly failing when data unexpectedly changes. 

The goal of \textbf{\textit{readr}} is to provide a fast and friendly way to read tabular data into R.  
\end{frame}
%==================================%
\begin{frame}[fragile}
\large
\textbf{readr}\\
The most important functions are:



\begin{itemize}
\item Read delimited files: \texttt{read_delim()}, \texttt{read_csv()}, \texttt{read_tsv()}, \texttt{read_csv2()}.

\item Read fixed width files: \texttt{read_fwf()}, \texttt{read_table()}.

\item Read lines: \texttt{read_lines()}.

\item Read whole file: \texttt{read_file()}.

\item Re-parse existing data frame: \texttt{type_convert()}.
\end{itemize}
